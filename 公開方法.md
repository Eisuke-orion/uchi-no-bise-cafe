# 無料で公開する方法

このサイトは `index.html` と `styles.css` と画像だけで動く静的サイトです。レンタルサーバーを借りなくても無料公開できます。

## おすすめ: GitHub Pages

費用: 無料

向いているケース: まず公式HPを公開したい、更新頻度が多すぎない、独自ドメインはあとで考えたい。

手順:

1. GitHubでアカウントを作成します。
2. 新しいリポジトリを作成します。
3. この `site` フォルダ内のファイルをアップロードします。
4. リポジトリの `Settings` → `Pages` を開きます。
5. 公開元を `main` ブランチ、フォルダを `/root` にして保存します。
6. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開されます。

公式情報:

- https://pages.github.com/
- https://docs.github.com/en/pages/getting-started-with-github-pages/what-is-github-pages

## もう一つのおすすめ: Cloudflare Pages

費用: 無料プランあり

向いているケース: 独自ドメインを使いたい、表示速度や将来の拡張も気にしたい。

手順:

1. Cloudflareのアカウントを作成します。
2. Pagesで新しいプロジェクトを作成します。
3. GitHubリポジトリを連携するか、ファイルをアップロードします。
4. ビルド設定は不要です。出力先は空欄または `/` で大丈夫です。
5. 公開後、必要なら独自ドメインを接続します。

公式情報:

- https://pages.cloudflare.com/
- https://developers.cloudflare.com/pages/functions/pricing/

## 独自ドメインについて

無料公開だけなら、GitHub PagesやCloudflare PagesのURLをそのまま使えます。

公式HPらしく見せるなら、将来的に `uchinobise-cafe.com` のような独自ドメインを取得して接続するのがおすすめです。ドメイン代だけは年額で費用がかかります。
